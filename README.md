# Lendsqr Frontend Assessment

##  Overview
A React (Next.js) application built to replicate the Lendsqr admin dashboard based on the provided Figma design from Lendsqr.

##  Tech Stack
- Next.js (React)
- TypeScript
- SCSS Modules
-Yup validation
- React Icons

##  Features
- Login page with validation
- Dashboard layout
- Users list (500 records)
- User details page with localStorage persistence
- Responsive design
